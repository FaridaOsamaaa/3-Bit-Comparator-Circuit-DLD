# 3-Bit Comparator Circuit

## Overview
This is a Digital Logic Design (DLD) academic project that implements a 3-bit magnitude comparator circuit.

The circuit compares two 3-bit binary inputs (A2 A1 A0 and B2 B1 B0) and produces three outputs indicating the relationship between the two numbers.

---

## Problem Statement
Design a comparator that receives two 3-bit inputs:

- A = A2 A1 A0  
- B = B2 B1 B0  

And produces three outputs:
- x = 1 if A = B
- y = 1 if A > B
- z = 1 if A < B

---

## Implementation
The circuit was designed using basic digital logic components.

It includes:
- Truth table generation
- Boolean expression derivation
- Logic gate implementation
- Circuit optimization

---

## Outputs
- x → Equality output (A = B)
- y → Greater than output (A > B)
- z → Less than output (A < B)
